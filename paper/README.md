# CAVA paper

This folder contains the LaTeX source for the CAVA research manuscript:

```text
CAVA: Canonical Action Verification and Attestation for Runtime Governance of Agentic AI Systems
```

Build locally with a standard TeX Live distribution:

```bash
latexmk -pdf -interaction=nonstopmode -halt-on-error main.tex
```

The public arXiv record is now available at:

```text
https://arxiv.org/abs/2607.13716
```
